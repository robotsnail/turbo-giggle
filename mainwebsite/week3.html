Position experiments

Below I try the different position values: static, relative, absolute, fixed, sticky, inherit. Scroll to see how they behave :)

static: the default. Element just follows the normal flow.
relative: I moved this box 20px down and 20px right with top and left, but it still keeps its original space.
absolute: taken out of the normal flow and positioned 10px from the top/left of the grey parent box.
fixed: this bar stays stuck to the bottom of the window even when you scroll.
sticky: scroll until this box reaches the top of the page; then it sticks there.

This parent has position: relative. The child below uses position: inherit, so it inherits the same positioning mode.

inherit: inherits position: relative from the parent.

HTML entities

HTML entities are special codes that let you display characters that normally have a meaning in HTML (like < or &). Here are the required ones:

  • & → written as &amp; → displays an ampersand.
  • " → written as &quot; → displays a double quote.
  • < → written as &lt; → displays a less-than sign.
  • > → written as &gt; → displays a greater-than sign.
  •   → written as &nbsp; → creates a non-breaking space (a space that will not collapse or wrap).
  • € → written as &euro; → displays the Euro symbol (€).
  • © → written as &copy; → displays the copyright symbol (©).
  • ™ → written as &trade; → displays the trademark symbol (™).

Below is a sentence using all of them:

© 2025 My Portfolio  |  Price: 25€  |  This is a trademarked name™  |  Symbols: & " < >
